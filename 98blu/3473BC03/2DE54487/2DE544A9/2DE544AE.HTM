INFOPUMP Workshop
La giornata di workshop comprende tutte le problematiche legate all'utilizzo di InfoPump, sviluppo degli script e gestione del sistema InfoPump. I partecipanti lavoreranno in un ambiente interattivo. Argomenti trattati:

• utilizzo di InfoPump Manager;
• creazione e monitoraggio di una richiesta di interscambio dati;
• creazione di script per manipolare e trasferire i dati;
• uso dello script-debugger;
• identificazione errori;
• InfoPump Server Administration.

Ai partecipanti è richiesta una conoscenza generale di programmazione C ed una familiarità con i maggiori database tipo SQL. La conoscenza del Basic è un plus.
